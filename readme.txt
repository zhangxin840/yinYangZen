R-ECO Solitaire是环保主题方块消除类游戏。
使用HTML5相关技术制作，打开网页即玩，不用Flash插件，可离线存储最高分。

为了更好的动画效果、字体与离线存储，推荐使用谷歌或火狐浏览器。


规则：

从手中出一种颜色若干方块到工厂。

工厂一端凑齐四个就消除并得分。

自动从垃圾堆收方块，垃圾堆填充的数量是工厂端方块数量+1。

也可以选择点击分数图标直接获得1分并从垃圾堆回收方块。

简单模式下，手中方块数量超过6个就会Gameover。
普通模式下方块数量超过5个Gameover。普通难度比简单难度有更多可拿的分数。

任意一堆分数拿完胜利，追加4分。


设计意图：

游戏规则根据网页单人游戏的特点做了很多改动，尽量给玩家更多选择并提高可重玩性。

点击分数直接得1分的规则，使得玩家可能主动将负分点掉，或者尽量避免将高分点成1分，
这种设计意图实现 "避免挑战只得到一个平庸的结果" 的哲学暗示。

设置胜利追加分是为了鼓励玩家坚持将游戏进行到底，挑战高分。
